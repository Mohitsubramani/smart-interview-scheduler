# Smart Interview Scheduler

An automated interview scheduling system that detects time conflicts and
optimally assigns interviewers to candidates.

## Features
- Conflict detection using interval scheduling
- Optimized interviewer allocation
- MySQL-based persistence
- Scalable backend design

## Tech Stack
- Java
- MySQL
- Data Structures (Greedy Algorithms)

## How it Works
1. Collects availability of candidates and interviewers
2. Detects overlapping time slots
3. Generates optimal interview schedule

- Web interface
